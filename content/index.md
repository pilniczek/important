## Page types

- **Tool** — Introduces a specific library, framework, or app.
- **How To** — Step-by-step instructions for getting something done or code snippets.
- **Approach** — A way of thinking / principles for writing code.
- **Course** — A pointer to learning material elsewhere.
- **People** — People-focused topics.
- **Grow** — Self-development / personal growth.
