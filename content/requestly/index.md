Tool to modify HTTP requests

[https://requestly.io/](https://requestly.io/)

Unlike Tweak, this tool works in 100% cases. Requestly has an “request targeting type” option where you can switch between “partial” and “exact” match. Tweak doesn’t have any such option. Tweak uses “partial” match as an default option [https://tweak-extension.com/docs/intro#url-and-label](https://tweak-extension.com/docs/intro#url-and-label) and you can enforce “exact” match only via “regexp” option.
