- **Windows / Linux** — [git-scm.com/install](https://git-scm.com/install).
- **macOS** — install via Homebrew: [`brew install git`](https://formulae.brew.sh/formula/git).

After installing, set up your [[configuration-example|Configuration Example]].

## Bundled Git vs system Git

IDEs (VS Code, Cursor) and tools like Claude can install their own Git binary that diverges from the system one. If your IDE shows no Git data after install, see the Troubleshooting section of [[git#troubleshooting|GIT]].
