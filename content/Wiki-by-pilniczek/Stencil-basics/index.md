`@Prop({reflect: true}) readonly variant: ButtonVariantType` 

The reflect!

```jsx
Purpose: Defines the style variation of the button.
Type: ButtonVariantType (a type alias for specific string values).
Reflect: The reflect: true option means that the property value will be reflected as an attribute in the DOM. This allows the property to be used in CSS selectors and observed for changes.
```
