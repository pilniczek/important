**Expecting for:**

Web design which can be easily made with webflow  (use webflow solution not custom)

**what can be a problem :**

1. complex animation  -  (smth like complex animated slider etc)
2. Working with dynamic data (API’s etc.) - (only by custom code )
3. Chat (support , chatbot) (idk)

**what will be a problem :**

1. Support for multilingual websites - 
2. Big Data  (Webflow provides a **4MB** limit on image size uploads)

**what will be cool:**

1. use webflow breakpoints  - 
2. similar design for desktop/laptop/mobile version 

**what needs discussion :**

GDPR ([https://discourse.webflow.com/t/european-gdpr-any-news/148422/14](https://discourse.webflow.com/t/european-gdpr-any-news/148422/14)**)**

It is Important to understand that every complex (which webflow can’t handle) decision will be made by custom code which means more time and problems with finding solution, maybe less performance 

We have no idea how does the figma-generated website looks like. It is hard to say what can be found there.
