What CMS we want to use and why?

### Sanity

without self-hosted variant

need to learn [**GROQ**](https://www.sanity.io/docs/groq)

### Strapi

self-hosted variant

not user friendly implementation to already existing projects (Strapi flow is - create tables in Strapi and the connect FE)

don’t connect to existing database - you will lost data https://github.com/strapi/strapi/issues/13732

### Directus

self-hosted variant

straightforward and fast CMS if you already have existing BE

Directus makes a mirror of your existing database, so you can instantly edit all your data from CMS

together with Supabase works like a Swiss knife

### Notion

not created as CMS in the first place

None of these CMSs can’t easily share same users for CMS and FE.
