Virtual lists enable to render (be part of a DOM) only the part of an app which is currently in view. This technique is useful for infinite scrolling such as social network dashboards (walls), where increasing size of DOM starts to cause performance issues.

The most popular virtual lists library is: https://github.com/bvaughn/react-virtualized

I don’t know why, but in the past I was excited about this library [https://virtuoso.dev/](https://virtuoso.dev/). Sadly, I can’t remember what it solves better than react-virtualized :(.
